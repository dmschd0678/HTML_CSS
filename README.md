### HTML/CSS 기본 문법 공부

## HTML

- **html:5 + tab** : 자동 완성
- **`<h?> 글씨 </h?>`** : ?의 숫자 크기가 작을수록 글씨 크기 커짐
- **`<a href = "링크">글씨</a>`** : 누르면 링크 이동(페이지 이동)
- **`<img src = "이미지 경로", alt = "">`** : 이미지 불러오기
- **`<br>`** : 줄 바꿈
- **`<table></table>`** : 테이블 만들기
    
    - `<th></th>` : 테이블 헤더 만들기
    - `<tr></tr>` : 테이블 행 만들기
    - `<td></td>` : 테이블 열 만들기

    - 표 디자인 : `<table ? = ""></table>`
        <table border = "1">
            <th>속성</th>
            <th>설명</th>
            <tr>
                <td> border</td>
                <td> 테두리</td>
            </tr>
            <tr>
                <td> bordercolor</td>
                <td> 테두리 색</td>
            </tr>
            <tr>
                <td> width/height</td>
                <td> 크기</td>
            </tr>
            <tr>
                <td> align</td>
                <td> 테이블 값 정렬</td>
            </tr>
            <tr>
                <td> colspan/rowspan</td>
                <td> 셀을 합침(행/열)</td>
            </tr>
        </table> 

- **`<ul></ul>`** : 순서가 없는 목록 만들기
- **`<ol></ol>`** : 순서가 있는 목록 만들기
    - `<li></li>` : list
- **`<p></p>`** : 문단 만들기
- **`<div></div>`** : 레이아웃 나누기
